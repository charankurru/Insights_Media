# Insights_Media
A website used to share insights and ideas with people . Basic media app developed using MeanStack application

This website is Developed using 
Angular 8
Nodejs
mongodb
express.js

FUNCTIONALITIES

1) It has the abitity to post newsFeed about any topic or project details
2) Anyone on the website can get  this post on their feed and can 
    => comment
    => like 
3) People can Follow others 
4) And the people can chat 
5) Reak time chattig is deveolped using Sockets.js
